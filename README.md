# WORLD MOST SECURE REGISTER SYSTEM 🔐 April Fool Edition 😁

*Delightfully useless, absurdly secure, and hilariously over-engineered.*

---

## About

This is a **satirical registration system** built for fun. It **never allows anyone to register successfully**, no matter what email or password you enter.

Features include:

* Dramatic **error messages** for every submission
* **Loud “faaah” sound effect** on failure
* **Chaotic, ugly, over-engineered UI**
* Instant clearing of inputs after “rejection”

It’s a playful parody of **over-engineered security systems** and **bad UX design**.
